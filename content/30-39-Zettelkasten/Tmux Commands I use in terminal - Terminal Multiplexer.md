### Tmux commands to use
create new tmux session
```bash
tmux
```

list tmux sessions!
```bash
tmux ls
```

attach to a tmux session
```bash
tmux a
```

Most of the time, I only need one tmux session.

```bash
Ctrl+B
```
This is the entry shortcut. If you type any letter after this, you get a command

```bash
Ctrl+B 1
Go to tab 1
```


```
Ctrl + B % 
Split horizontally

Ctrl+B "
Split Vertically
```
