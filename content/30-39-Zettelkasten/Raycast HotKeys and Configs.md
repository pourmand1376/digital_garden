Show Clipboard History: Hyper + V
Add Task (TickTick): Hyper + T
Search Bookmarks: Hyper + B
Create Note (Obsidian): Hyper + Q
Search Emojis: Hyper + ; (semi-colon)
Search Menu Items: Hyper + M
Search Snippets: Hyper + G
