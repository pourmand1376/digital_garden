من تگ رو یه جورایی مثل لینک می‌بینم. اما لینک‌هایی که تعریف شده هستند و تعدادشون محدوده. نکته‌اش هم همینه. 

در ثانی تگ برای Gathering و Retrieval استفاده میشه. یعنی اگر بخوایم دسته‌ای از نوت‌ها رو از سیستم استخراج کنیم. 

کارکردش مثل لینک نیست. 

این اقا از تگ صرفاً دو تا استفاده می‌کنه:
- وضعیت - State
- نوع-  Type

State is like consume, publish. 
Type is like blog, youtube, meeting and etc. 

طبق قانون برای خودم فقط به فایل‌هایی که تو پوشه Zettelkasten نیستند تگ می‌زنم. 

[Tags vs Folders](Tags%20vs%20Folders.md)
