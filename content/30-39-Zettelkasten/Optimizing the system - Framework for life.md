We want to make the system **simple**. Why?

Because Complexity is the hidden procrastination. 

But we don't want it to be very simple since it would be hard to process later. 

> [!tip]
> **Willpower is limited**. Micro-decisions cause decision fatigue.

We have to make choices. We want to make strategic choices not simple ones. 

We want to maximize the usage. 

We should have a **system** for daily choices.
