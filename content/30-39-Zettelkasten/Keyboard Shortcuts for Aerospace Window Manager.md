Move to Workspace 1: Alt + 1
Move window to Workspace B: Alt + Shift + B

Change Layout:
- Change to accordion: Alt + Comma
- Change to Tile: Alt + Slash

Change window size: Alt + Shift + Plus / Minus

Focus Another Window: Alt + H/J/K/L
Move Window in the workspace: Alt + Shift + H/J/K/L

Go into Service Mode: Alt + Shift + Semicolon
- Float window: Press F
- Join Window: Alt + Shift + H/J/K/L
- Reload Config: Escape
