ایده اینه که بتونی در کسری از ثانیه بین برنامه‌‌های مختلف جابه‌جا بشی. و این مستلزم اینه که بدونی هر برنامه‌ای رو کجا می‌گذاری. مثل این که ظرف‌ها رو تو آشپزخونه مرتب می‌کنیم و می‌دونیم کجاست. فقط می‌ریم برمیداریم. 

ایده مدیر برنامه‌ها اینه که روش مدیریت Floating Window Manager خیلی روش غیربهینه‌ای هست و میشه بهتر هم کار کرد. 

اول یک سری اصطلاح رو با هم فیکس کنیم:
- Workspace is used since Mac Spaces suck
- Each workspace contains some containers
- Containers can contain one or two apps

حالا قضیه اینه که برنامه‌ها به workspaceهای مختلف assign میشن و اینطوری «می‌دونی» که هر برنامه از قبل کجاست و نیازی نیست با Alt + Tab دنبالش بگردی. 

Some Rules:
- You don't need to resize windows
- You can delete the mouse

Each Container can have two properties:
- Layout (Tile, Accordion)
- Orientation (Horizontal, Vertical)

There are two modes in aerospace:
- Main Mode
- Service Mode

Main mode is the normal mode and service mode does several other things. Things that are rarely used like reloading configuration, make windows float, and join apps within containers. 

In order to do stuff you have to remember some stuff:
- How to Navigate Workspaces
- How to move app between workspaces 
- How to change layout
- How to change orientation
- How to focus on an app or container
- How to assign apps to workspaces 
- How to join apps within a container (at least 3 apps)
